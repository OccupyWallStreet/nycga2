== Description ==

Ever wanted to post status updates to your WordPress website, like you do on Facebook? Just insert the link and information is parsed to your site to show images and text from the target website. 

It looks like a Facebook status but it's on WordPress!

== Installation ==
= To Install: =

1.  Download the plugin file
2.  Unzip the file into a folder on your hard drive
3.  Upload the `/quick-status-lite/` folder to the `/wp-content/plugins/` folder on your site
4.  Visit Network Admin -> Plugins and Network Activate it there.
5.  Alternatively, you can activate this plugin on individual sites by going to the Site Admin -> Plugins menu and activating there.
