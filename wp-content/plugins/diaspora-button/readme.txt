== Description ==

Burnman's Diaspora Button plugin places a button at the end of your posts, allowing visitors to share it on the Diaspora* social network (JoinDiaspora.com) easily.


== Installation ==

1. Unzip and Upload the folder 'burnmans-diaspora-button' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
