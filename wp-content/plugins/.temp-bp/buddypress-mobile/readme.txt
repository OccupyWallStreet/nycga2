== Description ==
<bold>This version only for BuddyPress 1.5</bold>

Mobile BuddyPress theme optimized for viewing on iPhone, iPod touch. Will also work on Android and some Blackberry devices. Activate and plugin will detect mobile devices and use the BuddyPress Mobile theme. Options to switch between desktop and mobile theme. Two theme styles to choose from. Custom front page for the mobile site as well as a custom mobile menu. Option for visitors to get a add to home screen notice* and lazily loaded avatars fro faster page loads. 

Support forum located here: <a href="http://buddychimp.com">BuddyPress Mobile Support</a>


== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BuddyPress Mobile` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Choose various admin options.

= Manual Installation =

1. Upload `buddypress-mobile` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Choose various admin options.

== Frequently Asked Questions ==

= Can I edit the theme? =
Yes. In buddypress-mobile/themes/default you can edit the theme files.

= How to create a custom mobile homepage. =
Create a page in the WordPress admin titled bp-mobile. The mobile theme will use this page.
