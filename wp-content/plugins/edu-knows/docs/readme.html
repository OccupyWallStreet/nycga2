Themes Installations

  1. Unzip the dowloaded zip file.
  2. Upload the files inside themes folder to /wp-content/themes/.
  3. Make sure your install path are correct
    /www/var/site/wp-content/themes/thethemename
  4. Login to your dashboard » appearence and activate the themes
    * for wpmu user, you need to go to site admin and activate the theme before it showed in appearence
  5. Your theme should be usable and activated now

Plugins Installations

  1. Unzip the dowloaded zip file.
  2. Please follow the plugins install instruction and upload path.
  3. Make sure your plugin install path either one of this path
    /www/var/site/wp-content/plugins/
    or
    /www/var/site/wp-content/mu-plugins/
  4. Login to your dashboard » plugins and activate the plugin
  5. Your plugin should be usable and activated now
