Replaces the regular comment textarea with a lightweight full featured WYSIWYG Rich text editor. Also works with BuddyPress & bbPress.

== Description ==

This simple plugin will replace default plain comment textarea with a more <em>user-friendly, fully customizable and lightweight</em> <strong>"jWYSIWYG" WYSIWYG Rich Text Editor.</strong> 

Your commentators are definitely going to be love it because they will be able to comment even better with <strong>no mark-ups</strong> or without any knowledge about mark-up! 

This plugin is fast, lightweight, cross-browser enabled and very much user friendly. It has a full featured WYSIWYG Text Editor. 

Now this plugin also works with BuddyPress & bbPress. Therefore, it will provide WYSIWYG in Group forums post, reply and Private messege textarea in BuddyPress + Post & comment textarea in bbPress. 

After heavey optimization in version 0.8, customizations are a bit taugh but not impossible. If you really need to customize then find it at <a href="http://www.planetbd.net/rich-text-editor-for-comments/" target="_blank">Plugin Homepage. </a>

== Installation ==

1. Upload `texteditor-for-comments` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's all!
4. For customizations, see <a href="http://www.planetbd.net/rich-text-editor-for-comments/" target="_blank">at Plugin Homepage. </a>
5. This plugin can also be installed from Admin Panel.
