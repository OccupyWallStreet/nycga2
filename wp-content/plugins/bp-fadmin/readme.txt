This plugin brings site-wide-like administration options to the frontend, allowing group admins simpler management of all of their groups.  It can be further extended using the standard buddypress profile subnav registration functions (for example, the bp-wiki plugin hooks into this to enable movement of wiki pages between groups).

== Description ==

Included component allows group admins to easily promote/demote/ban members from all their groups.

It can be further extended using the standard buddypress profile subnav registration functions (for example, the bp-wiki plugin hooks into this to enable movement of wiki pages between groups).

== Installation ==

Download the plugin files to your plugin directory and activate.

Frontend administration can be accessed via a member's profile.  Each FA screen will only show a member groups that they are administrators of.
