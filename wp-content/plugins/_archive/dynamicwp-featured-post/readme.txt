== Description ==

DynamicWP Featured Post is a jQuery plugin that will list your featured post based the post tag and displayed as a widget. And the post will cycle automatically.

**Features**

* Easy to use
* A widget plugin
* Automatic image thumbnail
* Post will cycle automatically
* Unlimited post to display

**Demo**

You can check here : [DynamicWP Featured Post demo page](http://www.dynamicwp.net/featured-post-demo-page/).

**Credit** 

This plugin based [The jFlip Plugin](http://jquery.malsup.com/cycle/), author Mike Alsup

== Installation ==

1. Decompress the .zip archive and put the “dynamicwp-featured-post” category into your plugins directory (/wp-content/plugins/).
1. Or the other way, with your wordpress admin page > Plugins > Add New then choose upload on the top. Next choose File (dynamicwp-featured-post.zip) then Install Now. (see the screenshots)
1. Activate the plugin in the WordPress Plugins admin page.
1. Now you can see a new widget "Dynamicwp Featured Post" in your widget page. (see the screenshots)
