Provide all widgets with BuddyPress specific fields (conditional display logic)

== Description ==

After activating all current and possible future widgets will get extra BuddyPress specific options. 
You will be able to select on which users profiles or groups pages you want to display this widget and on which - don't.

So you will get more flexibility in arranging your sidebars (or widget-zones).

== Installation ==

1. Upload plugin folder `/bp-extend-widgets/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Widgets page in admin area and make customisations for each widget as needed
