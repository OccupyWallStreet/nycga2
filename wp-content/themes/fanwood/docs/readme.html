Fanwood

all-purpose, reponsive theme

Fanwood is a mobile-compatible, multi-purpose theme that features twelve optional layouts, seven menu locations, eighteen widget areas, and deep integration for many popular plugins like bbPress, BuddyPress, and Jigoshop.

Installing the theme through the uploader

You can install the theme through the built-in WordPress theme uploader.

  1. Download the fanwood.zip file to your computer.
  2. Go to Appearance > Themes > Install Themes in your WordPress admin.
  3. Click on the “Upload” link.
  4. This will take you to a page where you can upload the theme-name.zip file to your WordPress install.
  5. Once installed, you can activate the theme and use it for your WordPress installation.

Installing the theme via FTP

If you prefer to do things manually, this is the best option for you.

  1. Unzip the fanwood.zip file to your computer.
  2. Upload the fanwood folder to your wp-content/themes directory on your WordPress install using your preferred FTP software.
  3. Go to Appearance > Themes and activate the theme.

Theme Settings

Theme settings are available at Appearance > Theme Settings. Each setting will have a description telling you what it does and how to use it.

Plugin Integrations

A major part of Fanwood is deep plugin integration/support for popular plugins like bbPress, BuddyPress, and Jigoshop. Below are more information on what each plugin does.

Installing Fanwood Plugins

All recommended plugins can be found on the official WordPress.org plugins repository. To install them, log in to your WordPress administration panel and add each one via the Plugins > Add New page.

Recommended Jigoshop Settings
  • Jigoshop Images:
    • Tiny Images: 36x36
    • Thumbnail Images: 90x90
    • Catalog Images: 240x240
    • Large Images: 300x300
  • Cataglog Products Per Row:
    • Enter 2 or multiples of 2 if you're using any layout other than the full width layout.
    • Enter 3 if you're using the full width layout.
  • Catalog Products Per Page:
    • Enter a number that is a multiple of the Catalog Products Per Row number.
  • Known Issues:
    • Recent Reviews Widget does not display product ratings correctly. This is Jigoshop issue, not Fanwood issue.
    • Filter Price Widget throws an error when layered navigation returns only 1 product. This is a Jigoshop issue, not Fanwood issue.
