 BizzThemes Framework Library

 ## QUICK OVERVIEW

 Framework library contains all files that are used as theme backbone and power the overall
 theme performance. Without these files theme will not work, so any code change or file 
 removal will result in theme not functioning properly. We strongly recommend you to leave
 this folder intact.
 
 ## SUMMARIZED INSTRUCTIONS
 
	* UPDATE framework by simply overriding lib_frame folder with a new one
	* DO NOT modify any of lib_frame folder files
