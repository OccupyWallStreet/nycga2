How to apply languages:

1. In your wp-config make sure you have the following set to your language:
define('WPLANG', 'en_EN');
Ie; maybe fr_FR.. or whatever language you are using.

2. Then in your parent theme make sure you put your translated files in languages/.

3. Then make sure you save your language file as fr_FR or whatever country code you are using.
