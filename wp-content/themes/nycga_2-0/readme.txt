To install:
	•	Unzip and upload buddypress-widget-theme folder to your WP themes folder wp-content/themes/buddypress-widget-theme/ and activate.
	•	Create a new page called Home choose Wide Thin Thin or Thin Wide Thin as the page template.
	•	Then in side bar settings >> reading choose the home page you created as the static front page.
	•	Go to widgets and add widgets to the columns
This is the exact default BP 1.5 theme with a custom page and functions to allow a widgetized page.
